== Description ==
Add collapse and expand feature for repeater field instances for better sorting.

**This plugin requires both the [Advanced Custom Fields](http://wordpress.org/extend/plugins/advanced-custom-fields/) plugin (version 4.0+ only) *AND* [Repeater Field](http://www.advancedcustomfields.com/add-ons/repeater-field/) paid add-ons.**

= Localizations =
* English


== Installation ==
= Requirements =
* WordPress version 4.0 and later (tested at 4.2)
* PHP 5.3

= Installation =
1. Unpack the download-package
2. Upload the folder and all folder and files includes this to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
